PdfSetBookmarkTree

Description

Sets the bookmark tree for a PDF file or a set of PDF files.

The bookmark tree is defined thanks to an XSL-FO file whose root element must be fo:bookmark-tree. Please report to the W3C specification to learn how to specify the XSL-FO bookmark tree file.

Parameters

Attribute Description Required
file The PDF to set the bookmark tree. Yes, unless a nested resource collection element is used.
fofile XSL-FO file defining the bookmark tree (the root element must be fo:bookmark-tree). Yes
preservelastmodified Preserve the last modified time of PDF files. No; defaults to false.
failonerror If false, log a warning message, but do not stop the build, when the PDF file to process does not exist or an error occurs while setting the bookmark tree. No; defaults to true.

Parameters specified as nested elements

fileset or any other resource collection

Resource Collections are used to select groups of PDFs to process.

Examples

Set the bookmark tree for a single PDF

<pdfsetbookmarktree file="myfile.pdf" fofile="tmp/bookmarks.fo"/>

Set the bookmark tree for all PDFs in a directory and its sub-directories

<pdfsetbookmarktree fofile="tmp/bookmarks.fo">
  <fileset dir="pdf_dir" includes="**/*.pdf"/>
</pdfsetbookmarktree>
