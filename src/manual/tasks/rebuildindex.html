RebuildIndex

Description

Rebuild the search index (*.pdx) that is used by Adobe Reader to perform fast multi-documents search.

Notes: - This task relies on Adobe Acrobat to rebuild the search index. You must ensure a valid installation of Acrobat is present and that it includes the Catalog module (only available in the PRO suites on the latest versions released by Adobe).
- This task doesn't create the search index. The (empty) search index must be first created in Adobe Acrobat and may then be updated using this task when the referenced PDFs have changed.

Parameters

Attribute Description Required
index The search index file (*.pdx) to rebuild. Yes
startdelay The delay (in seconds) to wait for Acrobat to start. No,
Defaults to 5 seconds.
logerrormatch The regular expression the last line of Acrobat's log file should match to let BibAnt detect an error occurred while rebuilding the index. This attribute is mainly provided to support other locals than English-US/UK, in which case the error message output by Acrobat in its log file may differ. No,
Defaults to .+\WErr.+ to match English-US/UK locals.
keeplog Option to keep the log file generated by Acrobat while building the search index. No,
Defaults to false

Examples

The following example rebuild the index.pdx index file:

<rebuildindex index="index.pdx" />
