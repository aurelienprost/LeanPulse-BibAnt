PdfSetOpenAction

Description

Sets the JavaScript open action for a PDF file or a set of PDF files.

Parameters

Attribute Description Required
file The PDF to add the open action to. Yes, unless a nested resource collection element is used.
openaction The JavaScript open action. Yes, unless JavaScript is enclosed within tags.
preservelastmodified Preserve the last modified time of PDF files. No; defaults to false.
failonerror If false, log a warning message, but do not stop the build, when the PDF file to process does not exist or an error occurs while adding the open action. No; defaults to true.

Parameters specified as nested elements

fileset or any other resource collection

Resource Collections are used to select groups of PDFs to process.

CDATA section

To define a complex JavaScript script as the open action, you can embed the code in a <![CDATA[ ... ]]> section so you don't need to escape <, > & or other special characters.

Examples

Set the open action for a single PDF

<pdfsetopenaction file="myfile.pdf" openaction="app.alert('Hello World!');"/>
	
<pdfsetopenaction file="myfile.pdf">
  <![CDATA[app.alert('Hello World!');]]>
</pdfsetopenaction>

Set the open action for all PDFs in a directory and its sub-directories

<pdfsetopenaction openaction="app.alert('Hello World!');">
  <fileset dir="pdf_dir" includes="**/*.pdf"/>
</pdfsetopenaction>
