PdfEncrypt

Description

Encrypts a PDF file or a set of PDF files.

Parameters

Attribute Description Required
file The file to encrypt. Yes, unless a nested resource collection element is used.
ownerpassword The owner password. Yes; unless certfile is defined.
userpassword The user password. Yes; unless certfile is defined.
certfile The X.509 certificate file. No
keylength The number of bits for the encryption key. No; default is 40.
canassemble The assemble permission. No; default is true.
canextractcontent The extraction permission. No; default is true.
canextractforaccessibility The extraction for accessibility permission. No; default is true.
canfillinform The fill in form permission. No; default is true.
canmodify The modify permission. No; default is true.
canmodifyannotations The modify annotations permission. No; default is true.
canprint The print permission. No; default is true.
canprintdegraded The print degraded permission. No; default is true.
preservelastmodified Preserve the last modified time of PDF files. No; defaults to false.
failonerror If false, log a warning message, but do not stop the build, when the PDF file to encrypt does not exist or an error occurs while encrypting. No; defaults to true.

Parameters specified as nested elements

fileset or any other resource collection

Resource Collections are used to select groups of PDFs to encrypt.

Examples

Encrypt a single PDF

<pdfencrypt file="myfile.pdf" ownerpassword="ownerpass" userpassword="userpass"/>

Encrypt all PDFs in a directory and its sub-directories

<pdfencrypt ownerpassword="ownerpass" userpassword="userpass">
  <fileset dir="pdf_dir" includes="**/*.pdf"/>
</pdfencrypt>
