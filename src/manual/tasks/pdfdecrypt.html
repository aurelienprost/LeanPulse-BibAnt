PdfDecrypt

Description

Decrypts a PDF file or a set of PDF files.

Parameters

Attribute Description Required
file The file to decrypt. Yes, unless a nested resource collection element is used.
password The owner password to the PDF or for the certificate. Yes
keystore Keystore that holds certificate to decrypt the document. No; unless the document is encrypted with a certificate.
alias The alias to the certificate in the keystore. No; unless the keystore contains more than one certificate.
preservelastmodified Preserve the last modified time of PDF files. No; defaults to false.
failonerror If false, log a warning message, but do not stop the build, when the PDF file to decrypt does not exist or an error occurs while decrypting. No; defaults to true.

Parameters specified as nested elements

fileset or any other resource collection

Resource Collections are used to select groups of PDFs to decrypt.

Examples

Decrypt a single PDF

<pdfdecrypt file="myfile.pdf" password="pass"/>

Decrypt all PDFs in a directory and its sub-directories

<pdfdecrypt password="pass">
  <fileset dir="pdf_dir" includes="**/*.pdf"/>
</pdfdecrypt>
