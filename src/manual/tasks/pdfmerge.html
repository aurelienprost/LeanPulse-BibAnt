PdfMerge

Description

Merges PDF files to a single document.

Parameters

Attribute Description Required
tofile The output document. Yes
force Recreate the output document, even if it is newer than the source documents. No; defaults to false.

Parameters specified as nested elements

fileset or any other resource collection

Resource Collections are used to select groups of PDFs to merge.

Important Note: certain resource collection types such as fileset, dirset and files are undefined in terms of order. To merge the documents, it is essential to specify explicitly the order to finally get the result you were expecting. For unordered collections, you can explicitly specify the order by using the sort element.

Example

Merge all PDFs from a directory and its sub-directories in the resources' natural order

<pdfmerge tofile="alldocs.pdf">
  <sort>
    <fileset dir="pdf_dir" includes="**/*.pdf"/>
  </sort>
</pdfmerge>
