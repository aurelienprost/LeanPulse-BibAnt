AcrobatVersion

Description

Stores the Adobe Acrobat version installed on the computer (when used as task) or checks for a specific Acrobat version (when used as condition).

Note: This task will fail if Adobe Acrobat isn't installed on the local computer.

Parameters

Attribute Description Required (Task) Required (Condition)
atleast The version that Acrobat is at least. The format is major.minor. No One of these.
exactly The version that Acrobat is exactly. The format is major.minor. No
property The name of the property to set. Yes No (ignored)

Examples

Stores the installed Adobe Acrobat version in the property acrover.

<acrobatversion property="acrover"/>

Stores the Adobe Acrobat version in the property acrover if the Acrobat version is 10.0 or higher. Otherwise the property remains unset.

<acrobatversion property="acrover" atleast="10.0"/>

Sets the property acrobat-is-exact-10 if Acrobat 10.0 is installed. Neither 9.0 nor 10.1 would match.

<acrobatversion property="acrobat-is-exact-10" exactly="10.0"/>

Sets Acrobat10isInstalled if Acrobat 10.0 is installed.

<condition property="Acrobat10isInstalled">
  <acrobatversion exactly="10.0"/>
</condition>
