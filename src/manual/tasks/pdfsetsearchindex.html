PdfSetSearchIndex

Description

Sets the search index (*.pdx) for a PDF file or a set of PDF files.

Parameters

Attribute Description Required
file The PDF to set the search index. Yes, unless a nested resource collection element is used.
path The full or relative path to the search index (*.pdx). Yes
preservelastmodified Preserve the last modified time of PDF files. No; defaults to false.
failonerror If false, log a warning message, but do not stop the build, when the PDF file to process does not exist or an error occurs while setting the search index. No; defaults to true.

Parameters specified as nested elements

fileset or any other resource collection

Resource Collections are used to select groups of PDFs to process.

Examples

Set the search index for a single PDF

<pdfsetsearchindex file="myfile.pdf" path="pdf/index/bible.pdx"/>

Set the search index for all PDFs in a directory and its sub-directories

<pdfsetsearchindex path="pdf/index/bible.pdx">
  <fileset dir="pdf_dir" includes="**/*.pdf"/>
</pdfsetsearchindex>
